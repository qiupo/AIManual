# 探索性数据分析（EDA）

## 🔍 发现数据的秘密

EDA是理解数据的关键步骤。AI让探索分析更全面、更深入、更快速。

## EDA核心内容

### 1. 描述性统计

```
"对数据集进行描述性统计：

数据：[说明]

分析：
1. 中心趋势（均值、中位数）
2. 离散程度（方差、标准差）
3. 分布形态（偏度、峰度）
4. 分位数（25%、50%、75%）

提供Python代码和解释"
```


## 🎯 EDA检查清单

```
□ 数据概览
  - 形状和大小
  - 字段类型
  - 基本信息

□ 单变量分析
  - 分布特征
  - 统计量
  - 可视化

□ 双变量分析
  - 相关性
  - 散点图
  - 交叉表

□ 多变量分析
  - 交互作用
  - 分组分析
  - 模式识别

□ 异常检测
  - 离群点
  - 异常模式
  - 影响评估
```
