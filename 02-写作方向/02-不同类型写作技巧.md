# 不同类型写作技巧

AI擅长各种类型的写作。掌握不同场景的技巧，能让AI发挥最大价值。

## 📝 写作类型总览

### 按目的分类

```
1. 商务写作
   - 邮件、报告、方案、合同

2. 内容创作
   - 文章、博客、视频脚本、广告文案

3. 学术写作
   - 论文、研究报告、文献综述

4. 创意写作
   - 小说、诗歌、剧本、故事

5. 技术写作
   - 文档、手册、教程、API文档

6. 营销写作
   - 广告语、产品描述、推广内容
```

## 📧 商务写作

### 类型1: 邮件写作

#### 场景A: 请求类邮件

```
提示词模板：
"写一封请求邮件

收件人：[职位/关系]
请求事项：[具体请求]
语气：礼貌、专业
包含：
- 开场问候
- 清晰的请求
- 说明理由
- 表达感谢

字数：100-150字"
```

**实战示例**：

```
提示词：
"写一封向领导请假的邮件

情况：
- 时间：下周一到周三（3天）
- 原因：家人需要陪同去医院检查
- 承诺：会提前完成手头工作
- 联系方式：紧急情况可联系

语气：礼貌、专业、负责任"
```

**AI生成**：

```
主题：请假申请 - [你的姓名] - 1月20-22日

尊敬的[领导姓名]：

您好！

我想向您申请下周的休假，具体时间为：
1月20日（周一）至1月22日（周三），共计3天。

休假原因是因为家人需要陪同前往医院进行详细检查，
由于检查项目较多且需要家人在场，因此希望能在这几天
陪同照顾。对此给您带来的不便，我深感抱歉。

关于手头的工作，我承诺：
1. 本周五前完成所有紧急任务
2. 提前整理好工作文档和进度
3. 与[同事姓名]做好工作交接
4. 期间保持手机畅通，紧急情况可随时联系

感谢您的理解和支持，期待您的批准。

此致
敬礼

[你的姓名]
[日期]

附件：请假申请表
```

### 类型2: 报告写作

```
提示词模板：
"写一份[类型]报告

主题：[主题]
目的：[报告目的]
读者：[目标读者]

结构：
1. 执行摘要
2. 背景和目的
3. 主要发现
4. 数据分析
5. 结论和建议

要求：
- 专业但不晦涩
- 数据支撑
- 图表说明
- 可操作的建议"
```

**实战示例**：

```
提示词：
"写一份季度销售分析报告

时间：2024年Q4
数据：
- 总销售额：500万（同比增长15%）
- 主要产品：A产品占40%，B产品占35%
- 主要客户：企业客户占60%，个人客户占40%
- 主要挑战：新竞争对手进入市场

要求：
- 数据分析
- 趋势说明
- 竞争分析
- 下季度建议

格式：专业报告，包含图表说明"
```

## 📰 内容创作

### 类型1: 文章写作

#### 博客文章

```
提示词模板：
"写一篇博客文章

主题：[主题]
目标读者：[读者画像]
文章目的：[教育/娱乐/启发]

要求：
- 吸引人的标题（3-5个选项）
- 引人入胜的开头
- 清晰的结构
- 实用的内容
- 行动号召结尾

字数：[范围]
语气：[专业/轻松/幽默]
SEO关键词：[关键词列表]"
```

**实战示例**：

```
提示词：
"写一篇关于'远程工作效率提升'的博客文章

目标读者：刚开始远程工作的年轻人
痛点：在家工作效率低、容易分心

要求：
1. 10个实用技巧
2. 每个技巧包含：
   - 具体做法
   - 为什么有效
   - 实际案例
3. 字数：1500字
4. 语气：友好、鼓励
5. 包含个人经验分享
6. 标题要吸引人（给5个选项）

SEO关键词：远程工作、效率提升、在家办公"
```

#### 技术文章

```
提示词：
"写一篇技术教程

主题：如何使用[技术/工具]做[任务]
目标读者：[初学者/中级/高级]
难度：[1-10级]

结构：
1. 介绍（为什么要学）
2. 前置知识
3. 环境准备
4. 分步骤教程
5. 常见问题
6. 进阶学习方向

要求：
- 代码示例完整可运行
- 详细注释
- 截图说明（文字描述）
- 错误处理说明"
```

### 类型2: 视频脚本

```
提示词模板：
"写一个[类型]视频脚本

视频类型：[教程/影评/Vlog/解说]
时长：[分钟]
平台：[B站/YouTube/抖音]

要求：
- 开头钩子（5秒抓住注意力）
- 主体内容结构清晰
- 包含镜头指导
- 标注重点
- 结尾互动

格式：
[场景] 镜头描述
[画面] 视觉内容
[台词] 口播内容"
```

**实战示例**：

```
提示词：
"写一个科技产品评测视频脚本

产品：AI智能手表X1
视频类型：深度评测
时长：8-10分钟
平台：B站

内容要点：
1. 开场：产品外观展示（30秒）
2. 核心功能演示（3分钟）
   - 健康监测
   - 智能助手
   - 运动模式
3. 优缺点分析（3分钟）
4. 对比竞品（2分钟）
5. 购买建议（1分钟）
6. 结尾互动（30秒）

要求：
- 节奏紧凑
- 数据支撑
- 幽默风趣
- 引导互动"
```

## 💼 营销文案

### 类型1: 产品描述

```
提示词模板：
"为[产品]写产品描述

产品特点：[特点列表]
目标客户：[客户画像]
卖点：[核心卖点]

要求：
- 突出价值主张
- 解决客户痛点
- 语言有吸引力
- 包含行动号召
- 长度：[字数]"
```

**实战示例**：

```
提示词：
"为一款智能降噪耳机写产品描述

产品特点：
- 主动降噪技术
- 40小时续航
- 透明模式
- 舒适佩戴
- 触控操作

目标客户：
- 通勤族
- 学生党
- 办公室白领

- 专注工作学习
- 享受音乐
- 性价比高

要求：
- 场景化描述
- 情感化表达
- 包含3个版本：
  1. 简短版（50字）
  2. 标准版（150字）
  3. 详细版（300字）"
```

## ⚡ 写作技巧总结

### 技巧1: 明确写作类型

```
✅ "写一篇商务报告"
✅ "写一篇博客文章"
✅ "写一首现代诗"

不同类型，要求完全不同
```

### 技巧2: 指定目标读者

```
✅ "写给5岁孩子的科学解释"
✅ "写给CFO的财务报告"
✅ "写给编程新手的教程"

读者决定语气和深度
```

### 技巧3: 说明写作目的

```
✅ "目的是说服读者购买"
✅ "目的是教育读者"
✅ "目的是娱乐读者"

目的决定策略
```

### 技巧4: 提供具体要求

```
✅ 字数、结构、格式
✅ 语气、风格
✅ 包含/避免的内容
✅ 特殊要求
```

## 小结

- AI擅长各种类型的写作
- 不同类型有不同技巧
- 明确类型、读者、目的是关键
- 提供具体要求获得更好结果
- 人工审核和调整很重要
