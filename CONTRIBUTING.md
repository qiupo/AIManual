# 贡献指南

感谢你对AI使用手册的关注!我们欢迎各种形式的贡献。

## 🤝 如何贡献

### 报告问题

如果你发现了错误或有改进建议:

1. 创建Issue描述问题
2. 提供详细的信息
3. 如果可能,提供改进建议

### 提交内容

如果你想添加新内容或改进现有内容:

1. Fork本仓库
2. 创建你的分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启Pull Request

### 内容贡献指南

#### 写作风格

- ✅ 使用清晰简洁的中文
- ✅ 提供实际可用的示例
- ✅ 包含代码示例时添加注释
- ✅ 使用Markdown格式
- ✅ 保持章节结构一致

#### 内容类型

我们欢迎以下类型的贡献:

- **新章节**: 添加新的应用方向(如:数据分析、创意设计等)
- **内容补充**: 在现有章节添加更详细的内容
- **示例更新**: 提供更好的示例和案例
- **错误修正**: 修正事实错误或bug
- **格式改进**: 改善文档的可读性
- **翻译**: 翻译成其他语言
