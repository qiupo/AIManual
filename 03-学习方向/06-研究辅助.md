# 研究辅助

## 🔬 AI作为研究助手

研究不再只是教授和博士的专利。AI让每个人都能进行高质量的研究工作。

```
传统研究的挑战：
❌ 文献查找耗时
❌ 信息整理困难
❌ 数据分析复杂
❌ 写作表达艰辛
❌ 协作沟通不畅

AI辅助研究的优势：
✅ 快速文献调研
✅ 高效信息整合
✅ 辅助数据分析
✅ 提升写作质量
✅ 促进团队协作
```

## 🎯 研究的6个阶段

### 阶段1：选题与假设

```
目标：找到有价值的研究问题

AI能帮你：

✅ 评估研究可行性
✅ 优化研究假设
✅ 设计研究方案

提示词模板：
"我想做关于[大领域]的研究

## ⚠️ 研究常见误区

### 误区1：问题太大

```
❌ "AI对人类的影响"
✅ "AI对程序员就业市场的影响"

建议：问题要具体、可研究
```

### 误区2：数据不够

```
❌ 收集少量数据就下结论
✅ 确保样本量充足

建议：AI帮你做样本量计算
```

### 误区3：方法不当

```
❌ 用错研究方法
✅ 选择合适的方法
