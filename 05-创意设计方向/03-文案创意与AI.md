# 文案创意与AI

## ✍️ AI驱动的文案创作

从广告语到品牌故事，AI让文案创作更有创意、更高效。

## 广告文案

### 文案类型

```
"为[产品/服务]创作广告文案

产品信息：
- 名称：[产品]
- 卖点：[3个核心卖点]
- 目标受众：[人群]
- 品牌调性：[专业/轻松/高端]

请提供：
1. 10个创意标题
2. 5个完整广告文案
3. 不同风格版本
   - 情感型
   - 理智型
   - 幽默型
   - 紧迫型
4. 行动号召"
```

### 标题优化

```
"优化广告标题：

原标题：[标题]

问题：[点击率低]

请生成10个优化版本：
1. 紧迫感
2. 好奇心
3. 利益导向
4. 痛点直击
5. 社会证明
6. 稀缺性
7. 数据支撑
8. 对比反差
9. 情感共鸣
10. 趣味幽默"
```

## 社交媒体

### 微博/推特

```
"创作社交媒体内容：

主题：[主题]
平台：微博/推特
目标：[互动/传播]

请生成：
1. 吸引人的开头
2. 核心内容（140字内）
3. 相关话题标签
4. 互动引导
5. 配图建议

风格：[轻松/专业/幽默]"
```

### 微信公众号

```
"写公众号文章：

主题：[主题]
目标读者：[画像]
目标：[阅读量/转化]

请提供：
1. 吸引人的标题（10个选项）
2. 开头引入（5种方式）
3. 文章大纲
4. 核心内容
5. 结尾行动号召

要求：
- 2000字左右
- 结构清晰
- 案例丰富
- 可读性强"
```

## 品牌故事

### 品牌叙事

```
"为品牌创作故事：

品牌信息：
- 起源：[创立故事]
- 使命：[价值主张]
- 价值观：[核心理念]
- 成就：[里程碑]

请创作：
1. 品牌起源故事
2. 创始人故事
3. 用户故事
4. 产品故事
5. 愿景故事

要求：
- 真实感人
- 体现价值观
- 有传播力
- 记忆点深"
```

### Slogan创作

```
"为品牌创作Slogan：

品牌：[名称]
定位：[定位]
差异化：[独特价值]

请提供20个Slogan：
- 短句（2-8字）
- 易记
- 传达核心价值
- 有情感共鸣

经典案例参考：
- Nike: Just Do It
- Apple: Think Different
- 德芙: 纵享丝滑"
```

## 文案优化

### A/B测试

```
"创作文案A/B测试：

目标：[转化/点击]
内容：[产品/活动]

请提供3组对比文案：
每组2个版本

并说明：
- 版本A策略
- 版本B策略
- 预期效果
- 测试建议"
```

### 优化迭代

```
"优化现有文案：

原文案：[文案]

问题：
- 点击率低
- 转化率差
- 传播力弱

请优化：
1. 标题优化
2. 开头优化
3. 结构优化
4. 语言优化
5. 行动号召优化

说明优化理由"
```

## 🎯 文案技巧

### 技巧1：痛点直击

```
"直击用户痛点：

产品：[产品]
用户痛点：[列出]

请创作：
1. 痛点描述文案
2. 解决方案文案
3. 效果承诺文案

要求真实可信"
```

### 技巧2：故事化

```
"用故事讲述产品：

产品：[产品]
用户故事：[案例]

请创作：
1. 问题阶段（痛苦）
2. 发现产品（转折）
3. 使用产品（解决）
4. 结果展示（成功）

情感化表达"
```

### 技巧3：社会证明

```
"加入社会证明：

产品：[产品]
评价：[用户评价]

请创作：
1. 用户证言文案
2. 数据证明文案
3. 专家推荐文案
4. 案例展示文案

增强可信度"
```
