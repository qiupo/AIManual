# AI辅助创意设计入门

## 💡 创意新范式

AI让创意设计从"灵感爆发"变成"持续产出"。

## AI如何改变创意

### 传统创意流程

```
灵感枯竭 → 苦思冥想 → 试错修改 → 最终产出
（耗时数天到数周）
```

### AI辅助流程

```
需求输入 → AI生成创意 → 选择优化 → 快速产出
（数小时到1天）
```

效率提升：5-10倍

## 创意生成技巧

### 技巧1：发散思维

```
"请为[主题]生成20个创意方向

要求：
- 大胆创新
- 覆盖不同角度
- 简短描述
- 可视化说明

类别：
1. 功能创新
2. 视觉创新
3. 体验创新
4. 技术创新
5. 情感创新"
```

### 技巧2：组合创新

```
"组合以下元素创造新概念：

元素A：[描述]
元素B：[描述]
元素C：[描述]

请：
1. 两两组合
2. 三三组合
3. 颠覆组合
4. 每个方案说明创意点"
```

### 技巧3：跨界借鉴

```
"从[其他领域]借鉴创意：

应用到：[你的领域]

请：
1. 分析其他领域的成功案例
2. 提取核心概念
3. 跨界应用
4. 创新方案"
```

## 提示词模板

### 创意激发

```
"我需要为[产品/品牌/项目]设计[内容]

背景：[说明]
目标：[目标]
受众：[人群]

请生成：
1. 10个不同创意方向
2. 每个方向的视觉描述
3. 关键元素说明
4. 情感定位"
```

### 方案优化

```
"优化以下创意方案：

当前方案：[描述]

问题：[痛点]

请提供3个改进版本：
版本1：保守优化
版本2：激进创新
版本3：平衡方案

说明改动理由"
```

## 🎯 创意评估

### 评估维度

```
□ 原创性
- 新颖独特
- 打破常规

□ 相关性
- 符合目标
- 切合主题

□ 可行性
- 技术可行
- 资源可及

□ 吸引力
- 视觉冲击
- 情感共鸣

□ 记忆点
- 易于记忆
- 传播性强
```
