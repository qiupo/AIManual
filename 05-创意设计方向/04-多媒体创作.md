# 多媒体创作

## 🎬 AI驱动的多媒体内容

从视频到音频，AI让多媒体创作更简单。

## 视频创作

### 视频脚本

```
"为[产品/主题]创作视频脚本

类型：教程/广告/故事/解说
时长：[秒数/分钟]
平台：抖音/B站/YouTube

请提供：
1. 完整脚本
   - 场景描述
   - 人物对话
   - 镜头指导
   - 时长分配
2. 分镜脚本
3. 音乐建议
4. 字幕文本"
```

### 短视频系列

```
"策划短视频系列：

主题：[主题]
集数：10集
每集：1分钟

请规划：
1. 每集主题
2. 内容大纲
3. 脚本框架
4. 更新计划
5. 互动设计

保持连贯性"
```

## 分镜设计

### 故事板

```
"设计视频分镜：

视频主题：[主题]
关键场景：[列出]

请为每个场景设计：
1. 画面构图
2. 镜头运动
3. 场景转换
4. 时长估算
5. 音效建议

可视化描述"
```

### 动画脚本

```
"创作动画脚本：

动画类型：2D/3D/定格

请提供：
1. 详细脚本
2. 角色设计

```

## 音频创作

### 脚口播稿

```
"创作播客脚本：

主题：[主题]
时长：30分钟
风格：[轻松/专业/访谈]

请提供：
1. 开场白（吸引人）
2. 内容大纲
3. 主要观点
4. 案例故事
5. 互动环节
6. 结尾总结

口语化表达"
```

### 背景音乐

```
"描述背景音乐需求：

场景：[视频/播客/活动]
情绪：[欢快/紧张/温馨]
时长：[时长]

请描述：
1. 音乐风格
2. 乐器配置
3. 节奏速度
4. 情绪变化
5. 音乐生成工具建议（如Suno AI）"
```
