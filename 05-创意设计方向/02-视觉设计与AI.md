# 视觉设计与AI

## 🎨 AI驱动的视觉设计

从Logo到UI，AI让视觉设计更高效。

## Logo设计

### 设计流程

```
"为[品牌/公司]设计Logo

品牌信息：
- 名称：[名称]
- 行业：[行业]
- 定位：[定位]
- 受众：[受众]

请提供：
1. 10个Logo创意方向
2. 每个的视觉描述
3. 配色方案
4. 字体建议
5. 应用场景示意"
```

### 提示词技巧

```
细化需求：
"设计一个科技感Logo

要求：
- 简洁现代
- 蓝色系
- 抽象图形
- 适合移动端

灵感参考：
- Apple（简洁）
- Nike（抽象）
- Airbnb（创意）"
```

## 海报设计

### 设计要素

```
"设计活动海报：

主题：[主题]
时间地点：[信息]
目标：[吸引参加/销售产品]

请设计：
1. 版式布局
2. 主标题设计
3. 配色方案
4. 元素选择
5. 视觉层级"
```

### 文案+视觉

```
"设计促销海报：

产品：[产品]
优惠：[折扣]
卖点：[3个卖点]

请提供：
1. 吸引人的标题
2. 视觉焦点
3. 信息层次
4. 行动号召
5. 整体风格"
```

## UI设计

### 界面设计

```
"设计[App类型]的[页面]

功能：[功能说明]
目标用户：[用户画像]

请设计：
1. 页面布局
2. 交互流程
3. 视觉风格
4. 配色方案
5. 字体规范
6. 组件设计"
```

### 设计系统

```
"创建设计系统：

品牌：[品牌]
风格：[风格]

请定义：
1. 色彩系统
   - 主色
   - 辅助色
   - 功能色
2. 字体系统
   - 标题
   - 正文
   - 说明
3. 间距系统
4. 圆角规范
5. 阴影效果"
```

## 🎯 实战案例

```
任务：设计品牌VI

AI辅助流程：
1. 品牌分析
   - AI提取核心价值
   - AI分析竞品

2. 创意生成
   - AI生成100+Logo创意
   - AI提供配色方案

3. 方案筛选
   - 人工筛选最优3个
   - AI优化细节

4. 延展应用
   - AI生成名片/海报等
   - AI统一风格

时间：从1个月缩短到1周
```
