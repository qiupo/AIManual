## 📋 输出质量问题

### 问题1：回答不准确

```
症状：
- AI给出错误信息
- 事实性错误
- 逻辑混乱

原因：
- 知识截止限制
- 提示词不清晰
- 缺乏上下文

解决方案：

方案1：提供更多上下文
"请基于以下信息回答：
[详细背景信息]

问题：[具体问题]

如果不确定，
请明确说明而不是猜测"

方案2：要求验证
"请给出答案后，
说明：
1. 答案的确定性程度
2. 可能的错误
3. 建议的验证方式"

方案3：多轮验证
"请从不同角度分析这个问题，
并验证答案的准确性"
```

### 问题2：回答不够深入

```
症状：
- 只给出表面答案
- 缺乏深度分析
- 内容过于简单

解决方案：

方案1：明确深度要求
"请深入分析[主题]

要求：
- 不只说是什么，
- 更要说为什么
- 提供案例
- 分析影响
- 给出建议"

方案2：追问
"你刚才说了[内容]，
请进一步深入：
1. 底层原理是什么？
2. 有什么例外情况？
3. 如何应用到其他场景？"

方案3：指定角度
"请从以下角度深入分析：
- 技术角度
- 商业角度
- 用户角度
- 竞争角度

每个角度都要深入"
```

### 问题3：回答不够结构化

```
症状：
- 内容混乱
- 逻辑不清
- 难以理解

解决方案：

方案1：要求结构
"请按以下结构回答：
1. 核心观点
2. 支持论据
3. 案例说明
4. 总结建议"

方案2：使用标记
"请用以下格式回答：
- 标题：[标题]
- 要点1：[内容]
- 要点2：[内容]
- ...

确保层次清晰"

方案3：分步骤
"请分步骤说明，
每步：
1. 明确目标
2. 具体做法
3. 预期结果
4. 注意事项"
```

### 问题4：回答不符合预期

```
症状：
- 理解偏差
- 方向错误
- 文风不对

解决方案：

方案1：提供示例
"我要的是类似这样的回答：
[提供示例]

请按照这个风格和结构
回答我的问题：[问题]"

方案2：明确要求
"请按以下要求回答：
- 语气：[专业/轻松/正式]
- 长度：[具体字数]
- 角度：[从谁的角度]
- 格式：[列表/段落/表格]

问题：[问题]"

方案3：对话修正
"你刚才的回答不太对，
我的意思是：[澄清]

请重新回答"
```

## 🤔 理解问题

### 问题5：AI理解有偏差

```
症状：
- 理解错误意图
- 关注重点不对
- 偏离主题

解决方案：

方案1：背景优先
"先让我说明背景：
[详细背景]

我的问题是：[问题]

请基于这个背景回答"

方案2：明确目标
"我的目标是：[目标]

为了达到这个目标，
请：[任务]"

方案3：确认理解
"请先确认你理解了我的需求：

我的需求是：[描述]

请用你的话复述一下，
确保理解正确"
```

### 问题6：需要补充信息

```
症状：
- AI询问不清楚
- 缺少必要信息
- 无法继续对话

解决方案：

方案1：主动提供
"关于[任务]，
相关信息如下：
- 背景：[背景]
- 约束：[约束]
- 目标：[目标]
- 资源：[资源]

请基于这些信息[任务]"

方案2：分步确认
"这是大任务，
我们分步进行。

第1步先确认：
[具体问题]

回答后再继续"

方案3：让AI提问
"关于这个任务，
你需要什么信息才能完成？
请列出问题清单"
```

## ⚙️ 技术问题

### 问题7：输出被截断

```
症状：
- 回答不完整
- 中途停止
- 缺少结尾

解决方案：

方案1：分段要求
"这个内容很长，
请分3部分输出：
第1部分：[范围]
第2部分：[范围]
第3部分：[范围]"

方案2：要求继续
"请继续完成，
从刚才中断的地方开始"

方案3：缩短范围
"内容太多，
请只关注最重要的部分，
控制在[X]字以内"
```

### 问题8：响应太慢

```
症状：
- 生成速度慢
- 等待时间长

解决方案：

方案1：简化任务
"这个任务太复杂，
我们简化一下：
[简化版本]"

方案2：拆分任务
"分成多个小任务：
任务1：[简单任务]
任务2：[简单任务]
...

我们一个个来"

方案3：缩短输出
"请给出简洁版本，
不要太多细节，
只要核心内容"
```

### 问题9：格式问题

```
症状：
- 格式混乱
- 代码格式错误
- 特殊字符问题

解决方案：

方案1：明确格式
"请用以下格式输出：
[具体格式要求]

确保格式规范"

方案2：使用代码块
"请用代码块输出，
确保格式正确：

```格式
内容
```"

方案3：分步验证
"输出前请检查：
□ 格式正确
□ 无特殊字符错误
□ 代码可运行"
```

## 📊 效率问题

### 问题10：反复尝试才对

```
症状：
- 第一次回答不满意
- 需要多次尝试
- 效率不稳定

解决方案：

方案1：一次性给全信息
"这是所有信息：
[完整信息]

请基于此给出最佳答案，
不要遗漏"

方案2：使用更好模板
"这是经过验证的提示词模板：
[优秀模板]

按照这个模板格式来提问"

方案3：建立标准流程
"不要每次临时发挥，
使用标准流程：
1. [步骤1]
2. [步骤2]
3. [步骤3]"
```

### 问题11：需要大量调整

```
症状：
- AI输出需要大量修改
- 不如直接自己写
- 节省时间不明显

解决方案：

方案1：提高要求
"提高输出质量标准：
- 必须达到可直接使用
- 无需大幅修改
- 格式完全正确

达不到请重做"

方案2：提供更多指导
"这是我的期望：
[详细要求]

这是参考标准：
[示例]

请按这个标准输出"

方案3：优化提示词
"当前提示词效果不佳，
请帮我优化：

当前提示词：[提示词]
效果：[描述问题]

请给出改进版本"
```

### 问题12：无法复用好效果

```
症状：
- 某次效果好，但无法重复
- 不清楚哪里做得好
- 提示词丢失

解决方案：

方案1：记录提示词
"建立提示词库：
- 效果好的提示词
- 使用场景
- 效果评价

方便复用"

方案2：分析原因
"这次效果很好，
请分析为什么：
[成功的对话]

关键要素是什么？"

方案3：模板化
"把成功的对话
总结成模板：

模板名称：[名称]
使用场景：[场景]
提示词：[内容]
注意事项：[要点]"
```

```
问题：
会不会泄露隐私数据？

风险：
- 对话可能被记录
- 敏感信息风险
- 数据使用政策

保护措施：
✅ 不输入敏感信息
✅ 匿名化数据
✅ 了解平台政策
✅ 使用企业版（如需要）

禁止输入：
- 密码密钥
- 个人隐私
- 商业机密
- 未公开数据
```
